# Dessert Fridge Management System Using Queue in Java

This project is a Java application that manages desserts using a queue data structure.

## Features
- Add dessert
- Remove dessert
- View next dessert
- Display all desserts
- GUI interface using Swing

## Technologies
- Java
- Queue (LinkedList)
- Swing GUI

## How It Works
This system simulates a fridge where desserts are stored using a queue. 
The first dessert added is the first to be removed (FIFO principle).
